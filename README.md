# Simple todo list

This is the simple todo list app build with nextjs typescript redux and tailwindcss

## Get Started

Prerequisite

```
node ^v14.17.0
npm ^6.14.13
```

**Run locally as dev mode**

- npm run install
- npm run dev
- open http://localhost:3000

**Build Mode (production)**

- npm run install
- npm run build
- npm run start
- open http://localhost:3000
